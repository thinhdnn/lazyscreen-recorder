# LazyScreen Recorder

Simple screen recorder for lazy people.

## Why LazyScreen

LazyScreen Recorder is built for one job: start fast, record cleanly, and stop automatically when needed.

## Core Features

### 1) Basic Screen Recorder
Record your screen quickly with minimal setup.

### 2) Screen Recorder + Speech-to-Text
Record your screen and generate speech-to-text subtitles for your recording.

> **Note:** Speech-to-text uses the Soniox API.  
> You must purchase and provide your own Soniox API key to use this feature.

### 3) Live Recorder with Auto-Stop
Great for long meetings: keep recording hands-free and auto-stop when everyone has left and no audio activity remains.

## Quick Start

```bash
npm install
npm run dev
```

Or run directly:

```bash
./run.sh
```
